一个**高级全栈电影流媒体应用**，完全复刻 Netflix 界面与交互体验。

## ✨ 亮点功能
- 现代 Next.js 15 App Router + Server Components
- TypeScript 严格类型安全
- Tailwind CSS + 暗黑模式 + 流畅动画
- TMDB API 实时获取海量电影/剧集数据
- 英雄横幅、分类行、搜索、详情弹窗、收藏夹
- 完全响应式 + 移动端友好
- 生产级项目结构（易扩展）
